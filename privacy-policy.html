Quantify - Privacy Policy

Last updated: February 16, 2026

What Quantify Does

Quantify is a Chrome extension that finds prediction markets relevant to the content you're reading. It works on X/Twitter tweets and any webpage.

Data Collection

Quantify does not collect, store, or transmit any personal data to its developers. There are no analytics, no tracking, no accounts, and no servers operated by Quantify.

What Data Is Processed

When you activate Quantify on a page or tweet, the following happens locally and via third-party APIs:

  • Tweet or page text is sent to the Anthropic API using your own API key for query generation and relevance filtering.
  • Search queries (short keywords extracted from the content) are sent to public prediction market APIs: Manifold, Polymarket, Kalshi, and Metaculus.
  • Your Anthropic API key is stored locally in Chrome extension storage on your device. It is only sent to the Anthropic API and is never transmitted anywhere else.

Data Storage

All data is stored locally on your device using Chrome's built-in extension storage (chrome.storage.local). This includes:

  • Your Anthropic API key
  • Your selected model preference

No data is stored on any remote server. Uninstalling the extension removes all stored data.

Third-Party Services

Quantify sends requests to the following third-party APIs when you activate it. Each service has its own privacy policy:

  • Anthropic - LLM API for relevance filtering
  • Manifold Markets - Public prediction market API
  • Polymarket - Public prediction market API
  • Kalshi - Public prediction market API
  • Metaculus - Public prediction market API

Permissions

The extension requests the following permissions, each with a specific purpose:

  • activeTab - Read the current page content when you click the extension icon
  • sidePanel - Display market results in a side panel
  • storage - Store your API key and preferences locally
  • scripting - Inject the Q button on X/Twitter pages
  • Host permissions - Make API calls to market platforms and Anthropic

Changes to This Policy

If this policy changes, the updated version will be posted at this URL. The extension itself does not auto-update its privacy practices without a new version release.
